VK Radar

Десктопное приложение для мониторинга групп ВКонтакте. Автосканирование, поиск, уведомления. Всё в одном .exe файле.

РАДАР
Посты на радаре — перетаскивание, сонар, 14 тем оформления

Автосканирование

Автоматически проверяет все добавленные группы. Rate limiter встроен — VK API не блокирует.

Своё окно

Полноценное десктопное приложение на Tauri v2. Не браузер, не Electron — компактный нативный .exe.

Один файл

Всё в одном бинарнике: интерфейс, API-сервер на Axum, сканер VK. Никаких БД или Docker.

Приватность

Данные хранятся только у вас. Никаких серверов, никакой телеметрии. Всё работает локально.

Радар-интерфейс

Посты отображаются на анимированном радаре. Сонар, перетаскивание, 14 тем, частицы и небула-эффекты.

Гибкий импорт

Загружайте список групп из .txt/.csv. Экспорт групп по периодам: сегодня, неделя, всё время.

Как начать

1

Скачайте

Нажмите «Скачать .exe» или «Установщик». Файл ~13 МБ, ничего дополнительно ставить не нужно.

2

Запустите

Откройте radar.exe. При первом запуске появится окно с радаром — сканирование начнётся автоматически.

3

Добавьте группы

Откройте боковую панель (шестерёнка). Введите ссылку на группу ВК или импортируйте список из .txt/.csv.

4

Смотрите посты

Посты появятся на радаре. Перетаскивайте, включайте сонар (пробел), меняйте темы — всё в реальном времени.

Частые вопросы

Десктопное приложение для мониторинга постов из групп ВКонтакте. Отображает посты на анимированном радаре, автоматически обновляется, поддерживает уведомления.
Встроен по умолчанию. Просто скачайте и запустите — всё работает из коробки.
Ограничений нет. Сканер обрабатывает группы последовательно с rate limiter'ом (~3 запроса/сек), так что большое количество групп потребует больше времени.
В памяти приложения и JSON-файлах в папке ~/.radar/. Никаких внешних серверов или баз данных.
VK Radar написан на Rust + Tauri v2, а не на Electron. Весь интерфейс — нативный WebView2, а не браузерная прослойка. Итоговый .exe ~13 МБ против 100+ МБ у Electron-аналогов.
